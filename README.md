# xhs-content-ops-skill

Public, sanitized skill for Xiaohongshu content packaging and benchmark-based ops work.

This repository contains a reusable skill for:

- exported benchmark sample analysis
- title and cover optimization
- keyword embedding
- first-screen hook design
- posting format recommendation

## What it is based on

This public version was synthesized from anonymized internal working materials:

- an earlier `xhs-ops-optimizer` skill draft
- a partially completed reference draft focused on keyword clusters and ops heuristics
- anonymized Xiaohongshu export samples and analysis notes created on 2026-03-20

The public repo keeps the reusable methodology and removes personal creator strategy, local machine paths, and private workspace context.

Use it after a topic, outline, or draft already exists.
